Widget Chatbot SDK

Cách chạy demo:
  • Dev (npm run dev): dùng script type="module" bên dưới (file nguồn, không cần build).
  • Giống production: chạy npm run build, rồi chỉ bật thẻ dist/widget.js và tắt thẻ module (chỉ một thẻ script widget).

Nhúng thật trên site:

<script src="https://your-cdn/widget.js"
  data-public-key="YOUR_PUBLIC_KEY"
  data-api-url="https://your-api.example.com"></script>

data-api-url là gốc server (ví dụ http://localhost:8001); SDK tự thêm /api/v1 và gọi GET .../api/v1/chat/config.

Features

  • React-based UI (instead of Vanilla JS)
  • Shadow DOM for CSS isolation
  • Sales flow with cart, checkout, payment
  • Configurable branding (colors, logo, position)

API Endpoints (backend)

  • GET /api/v1/chat/config — cấu hình widget
  • POST /api/v1/chat — chat
  • POST /api/v1/chat/stream — streaming
