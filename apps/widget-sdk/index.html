Widget Chatbot SDK

Cách chạy demo:
  • Dev (npm run dev): chỉ bật khối A bên dưới (Vite biên dịch /src/main.tsx).
  • Giống nhúng production qua API: chạy cd apps/widget-sdk && npm run build, bật API (uvicorn cổng 8001), rồi chỉ bật khối B và tắt A. API phục vụ /sdk/chatbot-embed.js khi có file dist/chatbot-embed.js.
  • Không mở file này bằng Live Server để test B — dùng trang được Vite phục vụ hoặc site tenant thật.

Nhúng trên site tenant:

<script src="https://your-api.example.com/sdk/chatbot-embed.js"
  data-public-key="YOUR_PUBLIC_KEY"
  data-api-url="https://your-api.example.com"
  data-stream="true"></script>

data-api-url là gốc server (vd. http://localhost:8001); data-stream="true" bật SSE (POST .../chat/stream).

Features

  • React-based UI (instead of Vanilla JS)
  • Shadow DOM for CSS isolation
  • Sales flow with cart, checkout, payment
  • Configurable branding (colors, logo, position)

API Endpoints (backend)

  • GET /api/v1/chat/config — cấu hình widget
  • POST /api/v1/chat — chat
  • POST /api/v1/chat/stream — streaming
